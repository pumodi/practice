This is a D20 Dice rolling system built in Javascript. The purpose of this program is to help Dungeon Masters, or anyone who needs dice values from the D20 system, easily get dice rolls on various dice.
The program has the ability to roll D4, D6, D8, D10, D12, D20 & D100 dice as well as a dice with a number of sides specified by the user.

After selecting the number of dice to use the program will roll and calculate outputs based on the values of each dice rolled.


A user is able to get the following outputs:

   Sum of all dice
   Value of each dice rolled
   Highest dice rolled
   Lowest dice rolled
   Value of all dice rolled multiplied together
   Average value of all dice rolled


Currently, the program only calculates these outputs for one type of dice at a time. Meaning that you can't sum the value of all D4's with all the D20's.
This will possibly be a future in later updates.

Source code can be found at Github.
Feel free to fork the code and make your own D20 dice system.

If you get use out of this, please feel free to email me and let me know what you think.

Thanks for visiting and happy adventuring!
